\begin{frame}{Wrappers}
  However, in reality, \stdlist and \stddeque do NOT inherit from anything.
  There is no way to obtain a base class pointer that can point to both a list and deque.
  The solution is to wrap them with classes that DO shared a base class.
\end{frame}
