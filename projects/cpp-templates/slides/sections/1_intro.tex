\begin{frame}{Why C++ and why templates?}
  \begin{itemize}
    \item C++ is good for competitive programming and technical interviews.
    \item C++ is used for application development in the trading industry.
    \item Templates are useful for writing reusable library code.
    \item Become better at understanding library code.
  \end{itemize}

  Topic outline:
  \begin{itemize}
    \item \textbf{Intro to Templates} -- generic, reusable code
    \item \textbf{Type erasure/vtable dispatch} -- dispatch based on runtime type
    \item \textbf{move semantics} -- class design + resource management conventions
    \item \textbf{Variadic templates} -- functions and classes over arbitrary type lists
  \end{itemize}
\end{frame}
