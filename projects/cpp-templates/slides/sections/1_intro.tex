\begin{frame}{Why C++?}
  C++ is widely used in quantitative finance for performance-critical systems\\
  (e.g. execution engines, order books, and risk systems).\\

  \vspace{1pt}

  Strong C++ is commonly sought-after for developer/quant roles.

  \vspace{1pt}
  Today we will cover some powerful features in performant systems:
  \begin{itemize}
    \item \textbf{Templates} -- generic, reusable code
    \item \textbf{Type erasure/dynamic dispatch} -- flexible abstractions
    \item \textbf{Move semantics} -- efficient resource management
    \item \textbf{Variadic templates} -- functions and classes over arbitrary type lists
  \end{itemize}

  \begin{sqt:tip}[Goal]
    Develop a deeper understanding of modern C++ and the skills which companies are looking for.
  \end{sqt:tip}
\end{frame}
